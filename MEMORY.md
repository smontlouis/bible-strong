# Project Environment

- React Native: yes — Expo development client, Expo SDK 56.0.12, React Native 0.85.3.
- Package manager: Yarn 4.12.0.
- Metro: `yarn start`, default port 8081.
- iOS: `yarn ios`; development bundle ID `com.smontlouis.biblestrong.dev`.
- Android: `yarn android`; package `com.smontlouis.biblestrong`.
- Validation: `yarn test`, `yarn lint`, `yarn typecheck`.
- iOS native project and Podfile are present; Android is generated by Expo and has no checked-in native directory.
